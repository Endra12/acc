This file describes the framework ACC and other projects of mine are built upon.
It's assumed that you are already a flashable zip creator and/or Magisk module developer.

Tasks Executed By build.sh

1) Check shell syntax in all scripts.

2) Update version and versionCode in module.prop (values come from README.md > LATEST CHANGES).

3) Set the correct id (from module.prop) in install-*.sh acc/*.sh and bundle.sh.

4) Patch README.md.

5) Update busybox config (from acc/setup-busybox.sh) in $id/uninstall.sh and install scripts.

6) Unify installers for flashable zip (customize.sh and update-binary are copies of install-current.sh).

7) Generate flashable zips (main and uninstaller) and tarball (in _builds/).
